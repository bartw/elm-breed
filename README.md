# Elm Breed

I will use this repo to create a simple family tree app to learn [elm](http://elm-lang.org/).

## Install elm

```shell
npm install elm -g
```

## elm repl

With the elm repl you can test out elm code in your cli.

```shell
elm repl
```

```elm
:help
"hello"
"hello" ++ " world"
isNegative n = n < 0
isNegative 4
isNegative -3
point = { x = 3, y = 4 }
point.x
```
